# Aviral Shukla - Portfolio

A modern, interactive developer portfolio showcasing full-stack and AI-focused projects, skills, and contact details.

## Live Demo
- **BloodBond Platform:** https://bloodbond-platform.vercel.app/
- **Smart Reservation System:** https://smart-reservation-system.vercel.app/
- **Muggam:** https://muggam.vercel.app/

## Highlights
- Interactive **3D Tech Stack** visualization
- Animated **Skills Radar** with hover/click details
- Project cards with real-world problem/solution context
- Smooth navigation and section animations
- Responsive design for desktop/tablet/mobile
- Contact section with direct links (Email, Phone, LinkedIn, GitHub)

## Tech Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- D3.js (Skills Radar)
- Three.js (3D Tech Stack)
